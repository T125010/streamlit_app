産業別就業者数（男女比）分析アプリ
1. アプリの概要 

政府統計（e-Stat）のデータに基づき、日本における主要な小売・卸売業の男女就業者比率を可視化・分析するためのWebアプリです。 ユーザーが特定の業種を選択することで、その業界の性別構成を瞬時に把握し、労働環境の傾向を考察することができます。

2. 統計データについて 


・統計表名: 令和3年 経済センサス‐活動調査 
・データセット名/表番号: 卸売業，小売業に関する集計 / 表番号1 
・データ取得日: 2026年2月3日 
・各列の意味: 

・コード: 産業分類コード 
・産業名: 業種の名称 
・男性: 男性就業者数（単位：人） 
・女性: 女性就業者数（単位：人）

3. 使用したUI部品（未使用部品の活用）
授業で扱っていない、または機能的に意味のある以下の3つの部品を活用しました。
・st.metric:グラフでは読み取りにくい「総数」や「比率」をカード形式で強調し、直感的に把握させるため。
・st.divider:フィルタ設定や、指標とグラフの境界線を明確にし、視覚的な整理を行うため。
・t.download_button:ユーザーが分析した特定のデータをCSVとして持ち出し、再利用できるようにするため。

4. ギャラリー参照とアイデアの反映 

参考にしたアプリ/サイト: e-Stat「統計ダッシュボード」および Yahoo!ニュースのインフォグラフィック 
参考にした点: グラフだけでなく、主要なKPI（重要指標）を大きな数字で表示する構成。
反映内容: グラフの棒の上に具体的な人数を表示するアノテーション機能や、st.metric による女性比率のハイライトを実装しました。
